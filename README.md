## 🎯 Z-Image 使用技巧

1. **越詳細越好** - Z-Image 喜歡長而詳細的 prompt
2. **不需要 Negative Prompt** - 這個模型不使用負面提示詞
3. **可嵌入雙語文字** - 支援在圖片中生成文字
4. **建議長度** - 100-500 字效果最佳
